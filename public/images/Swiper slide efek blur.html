Biar next dan previous slide kelihatan blur di sisi kiri-kanan slide aktif di Swiper, lo bisa pakai kombinasi CSS scale, blur, dan Swiper class seperti .swiper-slide-prev dan .swiper-slide-next.

Berikut adalah langkah-langkah modifikasi kode lo:

✅ 1. Tambahkan efek blur dan scale ke slide non-aktif

Edit bagian SwiperSlide seperti ini:

tsx
<SwiperSlide className='swiper-slide-custom'>
  {/* ... */}
</SwiperSlide>

Ganti semua SwiperSlide pakai class swiper-slide-custom, biar bisa kita atur lewat CSS.

✅ 2. Tambahkan custom CSS untuk efek blur

Lo bisa tambahin ini di bagian global CSS atau inline via style tag, tergantung setup lo. Misalnya kalau pakai Tailwind, tambahin ke file global atau bisa pakai @layer Tailwind.

Kalau pakai Tailwind CSS (globals.css atau tailwind.config.css)

css
@layer utilities {
  .swiper-slide-custom {
    transition: all 0.3s ease;
  }
  .swiper-slide-prev,
  .swiper-slide-next {
    filter: blur(2px);
    transform: scale(0.95);
    opacity: 0.7;
  }
}

Catatan: swiper-slide-prev dan swiper-slide-next adalah class yang otomatis ditambahkan oleh SwiperJS ke slide sebelum dan sesudah slide aktif.

✅ 3. Atur centeredSlides ke true (opsional, tapi disarankan)

Biar slide aktif di tengah dan blur-nya di kiri-kanan:

tsx
<Swiper
  slidesPerView={1.3}
  spaceBetween={30}
  centeredSlides={true}
  pagination={{
    clickable: true,
  }}
  modules={[Pagination]}
  className="mySwiper h-72"
>

Hasil yang Didapat:

  • Slide aktif tetap jelas.

  • Slide sebelumnya dan setelahnya akan kelihatan blur, sedikit lebih kecil, dan transparan (opacity).

Kalau lo butuh versi tanpa Tailwind (misalnya pakai CSS biasa), tinggal pasang aja CSS biasa seperti ini:

css
.swiper-slide-custom {
  transition: all 0.3s ease;
}

.swiper-slide-prev,
.swiper-slide-next {
  filter: blur(2px);
  transform: scale(0.95);
  opacity: 0.7;
}

Kalau mau, gue juga bisa bantu bikin transisinya lebih dramatis atau interaktif, tinggal bilang aja gaya yang lo pengen.

1. Swiper Setup (responsive + custom class)

  }}

    1280: { slidesPerView: 3 },
  }}
  modules={[Pagination]}
  className="mySwiper h-72"
>
  <SwiperSlide className="swiper-slide-custom">...</SwiperSlide>
  <SwiperSlide className="swiper-slide-custom">...</SwiperSlide>
  {/* dan seterusnya */}
</Swiper>

2. Tambahkan Style untuk Blur & Scale

Kalau lo pakai Tailwind CSS, tambahkan ini di globals.css atau tailwind.config.css:

css
@layer utilities {
  .swiper-slide-custom {
    @apply transition-all duration-300 ease-in-out;
  }

  .swiper-slide-prev,
  .swiper-slide-next {
    filter: blur(2px);
    transform: scale(0.95);
    opacity: 0.7;
  }

  /* Optional: Buat slide yang lebih jauh makin blur */
  .swiper-slide-visible:not(.swiper-slide-active):not(.swiper-slide-prev):not(.swiper-slide-next) {
    filter: blur(4px);
    transform: scale(0.9);
    opacity: 0.5;
  }
}

Catatan:

  • .swiper-slide-visible = semua slide yang sedang kelihatan di layar.

  • Kita kasih styling berbeda supaya hanya prev dan next yang agak blur, dan sisanya bisa lebih blur kalau mau efek dramatis.

3. Kalau pakai CSS biasa (tanpa Tailwind)?

Masukkan ke file .css yang diimport:

css
.swiper-slide-custom {
  transition: all 0.3s ease-in-out;
}

.swiper-slide-prev,
.swiper-slide-next {
  filter: blur(2px);
  transform: scale(0.95);
  opacity: 0.7;
}

.swiper-slide-visible:not(.swiper-slide-active):not(.swiper-slide-prev):not(.swiper-slide-next) {
  filter: blur(4px);
  transform: scale(0.9);
  opacity: 0.5;
}

🧪 Bonus Tips:

  • Mau autoplay? Tambahin Autoplay ke modules dan konfigurasinya.

  • Mau efek lebih dramatis? Bisa tambahin brightness, grayscale, atau contrast.

ChatGPT said:

Oke, jadi sekarang maunya kalau tampil 3 slide sekaligus, yang tengah justru blur, dan yang kiri–kanan tetap tajam.

Secara default, Swiper pakai class .swiper-slide-active buat slide tengah, dan .swiper-slide-prev / .swiper-slide-next buat kiri–kanan.

✅ Solusi:

Kita cukup balik aturan styling-nya:

  • Yang aktif (tengah) → blur

  • Yang samping (prev & next) → tajam

1. 💡 Tambahin class di slide

Pastikan SwiperSlide kamu pakai class seperti ini:

tsx
<SwiperSlide className="swiper-slide-custom">...</SwiperSlide>

2. 🎨 Tambahin CSS Tailwind Custom (di globals.css atau tailwind.config.css)

css
@layer utilities {
  .swiper-slide-custom {
    @apply transition-all duration-300 ease-in-out;
  }

  /* Tengah (aktif) jadi blur */
  .swiper-slide-active {
    filter: blur(2px);
    transform: scale(0.95);
    opacity: 0.7;
  }

  /* Samping (prev dan next) tajam */
  .swiper-slide-prev,
  .swiper-slide-next {
    filter: none;
    transform: scale(1);
    opacity: 1;
  }

  /* Yang di luar prev/next, bisa diatur makin pudar */
  .swiper-slide-visible:not(.swiper-slide-active):not(.swiper-slide-prev):not(.swiper-slide-next) {
    filter: blur(4px);
    transform: scale(0.9);
    opacity: 0.5;
  }
}

  <SwiperSlide className="swiper-slide-custom">...</SwiperSlide>
  {/* dan seterusnya */}
</Swiper>

  • Kalau slidesPerView = 3 → slide tengah (active) blur, kiri–kanan tetap tajam.

  • Responsive, bisa adaptif di semua layar.

Kalau lo mau animasi tambahan (misal: blur + zoom + opacity fade), tinggal tambahin efek transisi aja.

⚠️ Potensi Masalah (dan Solusi)

1. Ukuran Container Swiper Tidak Cukup Lebar

Jika Swiper ditempatkan di <div> yang tidak fleksibel atau terlalu sempit (apalagi saat xl:flex-row), Swiper bisa kelihatan rusak: misalnya meluber, tersembunyi, atau tidak scrollable dengan benar.

✅ Solusi: Pastikan parent div Swiper punya class w-full atau flex-1, dan berada dalam layout yang memadai. Contoh:

tsx
<div className="w-full xl:w-1/2">
  <Testimoni />
</div>

Atau di komponen Testimoni, tambahkan style container:

tsx
<div className="w-full">
  <Swiper ... />
</div>

2. slidesPerView: 3.5 Membuat Slide Tengah "Blur" Tidak Terdeteksi

Kalau lo ingin memberi efek blur pada slide tengah saat slidesPerView = 3, lo harus identifikasi mana slide yang tengah secara aktif.

✅ Solusi: Tambahkan efek visual berdasarkan index aktif:

Gunakan state onSlideChange untuk menangkap slide aktif dan kasih style:

tsx
const [activeIndex, setActiveIndex] = React.useState(0);

<Swiper
  ...
  onSlideChange={(swiper) => setActiveIndex(swiper.activeIndex)}
>
  {data.map((item, index) => (
    <SwiperSlide
      key={index}
      className={`transition-all duration-300 ${
        index === activeIndex + 1 ? 'blur-sm scale-95 opacity-70' : ''
      }`}
    >
      {/* isi slide */}
    </SwiperSlide>
  ))}
</Swiper>

Atau kalau masih hardcoded, bisa cek index secara manual kalau cuma 3 slide tampil.

3. w-64 di SwiperSlide

Kelas w-64 bisa jadi terlalu sempit untuk slidesPerView > 2. Apalagi 3.5 slide berarti lebar total = 3.5 * 16rem = 56rem, which is 896px minimum. Pastikan container Swiper cukup.

✅ Coba ganti w-64 jadi w-full max-w-sm biar lebih fleksibel.

tsx
<SwiperSlide className='max-w-sm w-full border shadow-md rounded-4xl p-6 text-center'>

4. Pagination Tidak Klikable?

Lo set ini:

tsx
pagination={{
  clickable: false,
}}

Kalau butuh kontrol user, ganti jadi:

tsx
pagination={{
  clickable: true,
}}

5. Bug karena tidak ada loop: true

Jika jumlah slide sedikit dan ingin loop berulang, bisa tambahkan:

tsx
loop={true}

🔥 Tips Styling Blur Tengah (Bonus)

Kalau lo mau efek blur ke slide tengah saat 3 ditampilkan, lo bisa coba pakai kombinasi CSS dan custom class seperti ini:

tsx
<SwiperSlide
  className={`transition-all duration-300 ${
    index === activeIndex + 1 ? 'blur-sm scale-95 opacity-70' : ''
  }`}
/>

Jangan lupa pastikan Swiper diberi prop onSlideChange, dan lo tracking activeIndex.
